# 计算器插件开发计划

## 目标

为 ZTools 开发一个计算器插件，支持：
- 基本运算：加(+)、减(-)、乘(*)、除(/)
- 括号：()
- 百分比：%
- 计算历史记录
- 累计计算（连续计算）

## 当前上下文

- 项目模板：ZTools 官方示例 (`my-calculator`)
- 技术栈：React 18 + Vite + TypeScript
- 已安装依赖，dev server 运行在 `http://localhost:5173`
- 已有 Hello/Read/Write 三个示例功能

## 实现方案

### 4. 计算引擎设计

**状态管理：**
```typescript
interface CalculatorState {
  expression: string      // 当前输入表达式
  result: string          // 当前结果
  history: HistoryItem[]   // 计算历史
  hasResult: boolean       // 是否有可用结果（用于累计计算）
}

interface HistoryItem {
  expression: string
  result: string
  timestamp: number
}
```

**计算逻辑：**
- 使用逆波兰表达式 (Reverse Polish Notation) 算法解析表达式
- 支持运算符优先级：括号 > 百分比 > 乘除 > 加减
- 百分比处理：将其转换为小数 (例如 `50%` → `0.5`)
- 累计计算：当用户点击运算符时，如果已有结果，自动使用结果作为被操作数

**支持的表达式示例：**
- `100 + 200` → 300
- `100 * 20%` → 20
- `(100 + 200) * 2` → 600
- `100 + 200 =` (累计) `* 2` → 800

### 5. UI 设计

**布局：**
```
┌─────────────────────────┐
│  [显示历史] [清空历史]    │  ← 头部
├─────────────────────────┤
│  历史记录区域（可折叠）   │
├─────────────────────────┤
│                         │
│         0              │  ← 结果显示
│  100 + 200 = 300       │  ← 当前表达式
├─────────────────────────┤
│  AC  │  ()  │  %  │  ÷  │  ← 第一行
│   7  │   8  │   9  │  ×  │  ← 第二行
│   4  │   5  │   6  │  -  │  ← 第三行
│   1  │   2  │   3  │  +  │  ← 第四行
│   0  │   .  │   =  │     │  ← 第五行
└─────────────────────────┘
```

**按钮交互：**
- 数字/运算符：追加到表达式
- `=`：计算结果
- `AC`：清空当前表达式
- `()`：添加括号
- `%`：添加百分号
- 历史项点击：将结果填入表达式

## 风险与限制

- **除零错误**：需要捕获并显示 "Error"
- **表达式解析**：需处理非法表达式如 `++5`、`)(` 等
- **浮点精度**：JavaScript 浮点运算问题，如 `0.1 + 0.2 = 0.30000000000000004`，需四舍五入处理
- **长表达式**：限制表达式长度，防止 UI 溢出
